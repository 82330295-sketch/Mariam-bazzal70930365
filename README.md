# Community Help Hub

Community Help Hub is a ReactJS frontend web application that helps people request support, offer help, and connect with useful community services.

## Project Description

The project was created for CSCI390 Web Programming Phase 2. It focuses on solving a real-world community problem by allowing users to explore help categories, request assistance, and offer their own skills to support others.

## Features

- Responsive ReactJS frontend
- Multiple pages using React Router
- Home, About, Education, Home Services, Tech Support, Request Help, Offer Help, and Contact pages
- Request Help form with localStorage
- Offer Help form with dynamic cards
- Search and delete functionality
- Contact form
- Mobile-friendly design

## Technologies Used

- ReactJS
- React Router DOM
- JavaScript
- HTML5
- CSS3
- Git and GitHub
